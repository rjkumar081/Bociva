# MeraPopi - Digital Seva Center

Static React + Vite site (Tailwind CSS) ready for GitHub Pages.

## Quick start

1. Install dependencies:
```bash
npm install
```

2. Run dev server:
```bash
npm run dev
```

3. Build for production:
```bash
npm run build
```

4. Deploy to GitHub Pages:
- Create a GitHub repo named `MeraPopi-Digital-Seva-Center`
- Push this code
- In GitHub Pages settings, serve from `gh-pages` branch or `main`/`docs` depending on your setup.
- Alternatively deploy with Netlify/Vercel (they detect Vite).
